# Support

Use [GitHub Issues](https://github.com/reckerswartz/StreamBridge/issues) for reproducible bugs that do not contain private URLs, tokens, cookies, or credentials. Report security issues through the repository's [security instructions](https://github.com/reckerswartz/StreamBridge/security).

StreamBridge intentionally does not bypass DRM, anti-bot challenges, authentication, CORS restrictions, or private-header requirements. A detected stream may therefore be shown as unavailable outside its original page.
