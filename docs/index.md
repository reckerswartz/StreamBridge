# StreamBridge

StreamBridge detects HLS and direct-media requests after playback starts, verifies whether the URL can be reused without private browser credentials, and offers browser playback, copying, and platform sharing.

- [Privacy policy](privacy.html)
- [Support and known limits](support.html)
- [Source code and releases](https://github.com/reckerswartz/StreamBridge)

StreamBridge does not bypass DRM, replay authenticated headers, or collect analytics.
