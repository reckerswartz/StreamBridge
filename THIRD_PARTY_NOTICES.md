# Third-party notices

StreamBridge bundles unmodified release versions of these runtime libraries:

- [hls.js](https://github.com/video-dev/hls.js), Apache-2.0, used only in the extension-hosted HLS player.
- [webextension-polyfill](https://github.com/mozilla/webextension-polyfill), MPL-2.0, used for the cross-browser Promise-based extension API.

The full dependency inventory and licenses are recorded in the SPDX SBOM generated for every release. Build and test dependencies are not shipped in the browser packages.
